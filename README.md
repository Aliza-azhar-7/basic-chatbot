# ✨ Smart Study Bot Pro ✨

A beginner-friendly Python chatbot designed to help students stay productive, motivated, and engaged while studying.

The bot provides study tips, subject-specific guidance, fun commands, quizzes, motivational messages, and useful utilities such as a calculator, date/time display, and random generators.

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/yourusername/smart-study-bot-pro.git
```

2. Open the project folder:

```bash
cd smart-study-bot-pro
```

3. Run the program:

```bash
python smart_study_bot.py
```
